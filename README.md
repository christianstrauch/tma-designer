# TMA Designer
A simple browser based Tissue Microarray designer.
- Runs entirely in the browser
- Can run as a widget inside another page
- Takes a JSON input with specimens and (optional) TMA configuration, and delivers the TMA design as JSON

# Installation
You can test the TMA designer locally, e.g., using pnpm. 
Clone the repository to a local folder, and then, inside that folder, run
```bash
pnpm install
pnpm dev
```
Point your browser to http://localhost:3000
